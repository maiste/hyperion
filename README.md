# hyperion
 
## What is Hyperion ?
Hyperion is a Golang RESTApi server for the eos project and golactus.

## Build hyperion
To build Hyperion you can use the **make** command.

```sh
  # Build the project
  $ make

  # Equivalent to make
  $ make build

  # Build and run the project
  $ make run

  # Clean executable
  $ make clear
```
